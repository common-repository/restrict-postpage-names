== Description ==
This plugin restrict to create post or page names to your list of restricted names.

The restricted names can be configured from admin end "Restrict Post/Page Names" menu.

List down your restricted names( Separated by commas(,) ).

== Installation ==

Install this plugin is very simple:

1.	Upload the plugin's folder to the `/wp-content/plugins/` directory
2.	Activate the plugin through the 'Plugins' menu in WordPress
3.	That's it! Go to "Restrict Post/Page Names" and list down your restricted names( Separated by commas(,) ).
